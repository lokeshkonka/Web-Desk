# WebDesk UI Design

Theme:

Cozy Retro

Inspired by:

* Stardew Valley
* Coffee Talk
* VA-11 HALL-A
* Cozy pixel rooms

---

# Philosophy

Modern functionality with retro atmosphere.

Avoid:

* Glassmorphism
* Neon cyberpunk
* Windows clones
* Heavy blur
